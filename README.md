# FitSplit

**Personalized Workout Split Planner**  
A beginner-friendly React Native app that generates a personalised weekly workout routine based on your available gym days.
Built with Expo and designed to be clean, intuitive and responsive.

---

## Overview

FitSplit helps users plan their workout schedules efficiently by recommending the optimal split based on the number of days they can commit to the gym.

- Input your weekly availability.
- Receive a tailored workout split.
- View a visual weekly timeline of your routine.
- Easy to navigate, beginner-friendly UI.

---

## Features

- **Dynamic Recommendations:** Workout split suggestions based on 3–6 gym days.
- **Weekly Routine View:** See your weekly workouts laid out with clear indicators.
- **Reusable Components:** Progress dots and buttons built for consistency across screens.
- **Centralized Data:** Options, recommendations, and routines stored in a single `data.js` for maintainability.
- **Responsive Layout:** Mobile-first design using flexbox and Expo LinearGradient backgrounds.
- **Navigation:** Smooth page transitions via `expo-router`.

## Tech Stack

- **React Native** – Core framework for building mobile apps
- **Expo / Expo Router** – Simplified navigation and build tooling
- **Expo LinearGradient** – Gradient backgrounds for UI polish
- **JavaScript** – App logic and state management
- **React Hooks** – `useRouter`, `useLocalSearchParams` for dynamic routing

## Installation / Setup

1. Clone the repository
2. Install dependencies 'npm install'
3. Start the project 'npx expo start'
4. Open (Expo Go on your mobile device)
